# 15.3 Configure IP Default Static Routes

## Default Static Route

**default static route** — route, которая используется, когда в routing table нет более specific match. #networkterm

Ключевые записи:

- IPv4: `0.0.0.0 0.0.0.0`
- IPv6: `::/0`

![](https://itexamanswers.net/wp-content/uploads/2020/06/Default-Static-Route.jpg)

Default route особенно полезна на edge router, который знает:

- локальные сети;
- и один путь “во всё остальное”, например к ISP.

## Configure a Default Static Route

Примеры:

```bash
ip route 0.0.0.0 0.0.0.0 172.16.2.2
ipv6 route ::/0 2001:db8:acad:2::2
```

`ip route 0.0.0.0 0.0.0.0 172.16.2.2` #ciscoIOScommand
Создаёт IPv4 default static route через next hop.

`ipv6 route ::/0 2001:db8:acad:2::2` #ciscoIOScommand
Создаёт IPv6 default static route через next hop.

![](https://itexamanswers.net/wp-content/uploads/2020/06/Configure-a-Default-Static-Route.jpg)

## Verify a Default Static Route

Проверка:

```bash
show ip route
show ipv6 route
```

В output ищи:

- `S*` для IPv4 default route;
- `S` запись с `::/0` для IPv6;
- иногда строку про `gateway of last resort`.

> [!important] Самая важная мысль
> Default route не заменяет specific routes.
> Она используется только тогда, когда router не нашёл более точного совпадения.

> [!success] Если понял тему
> Ты можешь объяснить, что такое route of last resort, и настроить default static route для IPv4 и IPv6.
