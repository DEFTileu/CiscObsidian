# 15.0 Introduction

Module 15 — это уже не просто routing concepts, а реальная ручная маршрутизация.

Если Module 14 объяснял, как router мыслит, то Module 15 показывает, как мы сами добавляем нужные routes в routing table.

> [!abstract] Суть модуля
> `No dynamic learning` -> `administrator manually adds the route`

## Что нужно понять

После модуля ты должен уверенно объяснить:

- что такое static route и когда она полезна;
- чем отличаются `next-hop`, `directly connected` и `fully specified` static routes;
- как настраиваются IPv4 и IPv6 static routes;
- как настраивается default static route;
- что такое floating static route;
- что такое host route и зачем она нужна.
