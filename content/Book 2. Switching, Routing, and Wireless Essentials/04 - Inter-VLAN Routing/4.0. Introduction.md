# 4.0 Introduction

## 4.0.1 Why should I take this module?

Теперь ты уже умеешь делить switched network на `VLANs`.

Но здесь сразу появляется важный вопрос:  
как host из одной VLAN будет общаться с host из другой VLAN?

Именно это и решает **inter-VLAN routing**.

**inter-VLAN routing** — процесс маршрутизации traffic между разными VLANs. #networkterm

Пока devices находятся в одной VLAN, они общаются на Layer 2.  
Как только нужно выйти в другую VLAN, уже нужен Layer 3 device:

- router;
- или Layer 3 switch.

> [!abstract] Главная идея модуля
> `VLANs separate traffic at Layer 2`  
> `Inter-VLAN routing reconnects them at Layer 3`
