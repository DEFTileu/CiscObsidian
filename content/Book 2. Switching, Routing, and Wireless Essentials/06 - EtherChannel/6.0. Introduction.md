# 6.0 Introduction

`EtherChannel` решает очень практичную проблему: одного physical link уже мало, а несколько parallel links без специальной технологии STP просто заблокирует.

То есть идея такая:

- нам нужна **больше bandwidth**;
- нам нужна **redundancy**;
- но мы не хотим получить Layer 2 loop.

Именно здесь появляется EtherChannel.

**EtherChannel** — технология link aggregation, которая объединяет несколько physical Ethernet links в один logical link. #networkterm
**link aggregation** — объединение нескольких физических каналов в один логический. #networkterm
**port channel** — логический interface, который получается после объединения физических ports в EtherChannel. #networkterm

> [!abstract] Суть модуля
> `EtherChannel = multiple physical links, one logical link`
