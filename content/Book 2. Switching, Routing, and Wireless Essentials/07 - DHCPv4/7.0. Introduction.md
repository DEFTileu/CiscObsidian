# 7.0 Introduction

`DHCPv4` снимает одну из самых скучных и частых ручных задач в сети: раздачу IPv4 addresses клиентам.

В маленькой сети ещё можно настроить всё руками. Но как только devices становится много, static addressing превращается в источник ошибок и лишней работы.

**DHCP (Dynamic Host Configuration Protocol)** — протокол, который автоматически выдаёт IP parameters клиентам. #abbreviation
**DHCPv4** — версия DHCP для IPv4 networks. #networkterm

> [!abstract] Суть модуля
> `DHCPv4 = automatic IPv4 addressing`
